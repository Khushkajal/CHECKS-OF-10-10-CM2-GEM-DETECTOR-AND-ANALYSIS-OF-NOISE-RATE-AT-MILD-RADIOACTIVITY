# CHECKS-OF-10-10-CM2-GEM-DETECTOR-AND-ANALYSIS-OF-NOISE-RATE-AT-MILD-RADIOACTIVITY
This work can be used as a manual or basline material before working on GEM detector. This work was done for completion of M.Sc. (hons.) Physics at Panjab University, Chandigarh under the supervision of Prof. Vipin Bhatnagar and Asst. Prof. Sushil Singh Chauhan.

### Work Included:
- GEM foil handling and detector assembly
- High voltage (HV) stability testing
- Gain and efficiency characterization
- IV (Current-Voltage) curve analysis
- Noise and leakage current studies
- Detector quality control procedures

### Experimental Setup:
- Gas system handling (Ar/CO₂ mixtures)
- Signal readout using electronics and DAQ systems
- Data acquisition and monitoring

### Outcomes:
- Studied detector response under varying voltage conditions
- Analyzed detector stability and performance
- Contributed to understanding detector reliability for HEP experiments
